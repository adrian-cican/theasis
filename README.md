# Theasis

Repository for process data analysis, feature engineering, anomaly detection experiments, and research notes.

## Structure

- `data/`: raw, processed, and external datasets.
- `notebooks/`: numbered analysis workflow notebooks.
- `src/`: reusable Python modules.
- `experiments/`: reproducible experiment folders with notes, params, metrics, outputs, plots, and models.
- `reports/`: figures and written notes.
- `configs/`: project configuration.
- `logs/`: execution logs.
- `research/`: papers, summaries, notes, and ideas.
